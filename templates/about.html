This application uses a comprehensive heart disease dataset to predict cardiovascular risk. The dataset includes various health parameters such as:

About the AI Model

Our prediction model uses a Random Forest Classifier, a powerful machine learning algorithm that:

  • Considers multiple health factors simultaneously
  • Identifies complex patterns in health data
  • Provides risk probability rather than just binary classification
  • Maintains high accuracy across different demographic groups
